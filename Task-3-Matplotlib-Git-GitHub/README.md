# Task-3 Matplotlib, Git & GitHub

This task contains programs and practice exercises related to data visualization using Matplotlib and version control using Git & GitHub.

## Topics Covered

### Matplotlib

The Matplotlib section includes beginner-level data visualization programs such as:

* Line plot
* Bar chart
* Scatter plot
* Pie chart
* Histogram

These programs help in understanding how to represent data visually using Python.

* Branching and merging

The purpose of this task is to understand version control, collaborative development, and basic data visualization techniques.
