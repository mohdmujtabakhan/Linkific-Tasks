# Task-2 Python Basics

This task contains beginner-level Python programs created to understand the fundamentals of Python programming.

The programs included in this task cover:

* Variables and data types
* Input and output operations
* If-else conditions
* Loops
* Functions
* Lists and dictionaries
* Basic calculator program

The purpose of this task is to build a strong foundation in Python programming through hands-on practice and problem-solving.
