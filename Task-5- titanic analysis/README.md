# Titanic Dataset Analysis

## Project Overview

This project analyzes the Titanic dataset using Pandas and Matplotlib.

The objective is to explore passenger information and identify factors that influenced survival during the Titanic disaster.

---

## Tools and Libraries

- Python
- Pandas
- Matplotlib

---

## Analysis Performed

### 1. Who Survived More: Men or Women?

The dataset was filtered to include only surviving passengers.

A bar chart was used to compare the number of survivors by gender.

### Result

Women survived significantly more than men.

This supports the historical evacuation strategy of prioritizing women and children.

---

### 2. Average Age by Passenger Class

Passengers were grouped by travel class.

The average age was calculated using the mean() function.

### Result

First-class passengers generally had a higher average age than passengers in second and third class.

---

### 3. Survival Rate by Embarkation Port

Passengers were grouped by embarkation port.

The survival percentage was calculated and visualized.

### Result

Passengers embarking from Cherbourg showed a higher survival rate compared to Southampton and Queenstown.

---

## Key Insights

### Insight 1

Gender had a significant impact on survival.

Female passengers had a much greater chance of survival.

### Insight 2

Passenger class influenced passenger demographics.

First-class passengers tended to be older.

### Insight 3

Embarkation location showed differences in survival rates.

Cherbourg passengers generally experienced better survival outcomes.

### Insight 4

Social and economic factors likely affected survival chances during the disaster.

---

## Conclusion

The Titanic dataset reveals that survival was influenced by:

- Gender
- Passenger Class
- Embarkation Port

Using Pandas for data analysis and Matplotlib for visualization helped uncover meaningful patterns and insights from the dataset.
